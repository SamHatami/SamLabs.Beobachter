# SamLabs.Beobachter

SamLabs.Beobachter is a full rebuild of the original Log2Console, implemented as a modern desktop application on .NET 8 with Avalonia UI.

## Download
Latest release:
https://github.com/SamHatami/SamLabs.Beobachter/releases/latest

## Runtime
Windows release binaries are currently published as single-file, framework-dependent builds.

Required runtime:
.NET 8 (Desktop Runtime)
